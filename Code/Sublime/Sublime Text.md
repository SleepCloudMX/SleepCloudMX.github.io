# Sublime Text



### 1	Introduction

| Keys                      | Descriptions                     |
| ------------------------- | -------------------------------- |
| Ctrl + D                  | Next occurence                   |
| Shift + right click drag  | Column selection                 |
| Ctrl + Shift + L          | Split selection into lines       |
| home, end (Or PgUp, PgDn) | Move cursors to begining of line |
| Alt + Shift + W           | Wrap selection with tag          |
| Ctrl                      | Hide the mouse                   |
| Ctrl + W                  | Quit                             |
