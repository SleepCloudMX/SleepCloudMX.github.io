### A.4  常用积分

#### A.4.1  Dirichlet 积分

$$
\begin{align}
\Sa(x) &:= \dfrac{\sin x}{x} = \Si'(x), \\
\Si(x) &:= \int_0^x \dfrac{\sin t}{t} \dt.
\end{align}
$$


$$
\begin{align}
\intoi \Sa(x) \dx &=
\intoi \dfrac{\sin x}{x} \dx
= \intoi \dx \intoi \e^{-xy} \sin x \dy
\\
&= \intoi \dy \intoi \e^{-xy} \sin x \dx
= \intoi \dfrac{\dy}{1 + y^2} = \dfrac{\pi}{2}.
\end{align}
$$

$$
\begin{align}
&\hspace{1.3em} \intoi \intoi \Sa(x) \Sa(x + y) \dxdy
\\
&= \intoi \Sa(x) \dx
\int_x^{+\infty} \Sa(u) \dd{u}
\\
&= \intoi \Sa(x) \dx \pqty{
	\intoi \Sa(u) \dd{u} -
	\int_0^x \Sa(u) \dd{u}
} \\
&= \dfrac{\pi^2}{4} -
\intoi \Si(x) \Si'(x) \dx
= \dfrac{\pi^2}{8}.
\end{align}
$$

### A.5  三种卷积

**定义**

- **函数的卷积积分**	$ (f * g)(t) = \intoi f(t - s) g(s) \ds $.
- **序列的卷积和**	$ (f * g)[n] = \dsum_{i = -\infty}^{+\infty} f[n - i] g[i] $.
- **狄利克雷卷积**	$ (f * g)(n) = \dsum_{d \mid n} f\pqty{\dfrac{n}{d}} g(d) $.

**备注**

- 在概统里还会有分布的 R-S 卷积积分 $ (F * G)(x) = \inti F(x - u) \dd{G(u)} $, 如果能用 Riemann 积分表示, 则有 $ (F*G)(x) = \dint_{-\infty}^x (f * g) (t) \dt $, 即化为函数的卷积积分, 这里不再讨论. $ %为了以示区分, 序列的卷积和用中括号, 其余用小括号; 函数的卷积积分的自变量是连续的, 其余是离散的; 分布的卷积积分中, 函数均使用大写字母表示. $

- 在计算机科学里, 有时也会上述序列的卷积和的结果的一部分定义为序列的卷积, 这里也不再讨论.

**性质**

这三种卷积都有如下性质:

- 交换律 $ f * g = g * f $.
- 结合律 $ (f * g) * h = f * (g * h) $.
- 分配律 $ f*(g+h) = f*g + f*h $.
- 数乘 $ a (f * g) = (af) * g $.
- 平移
  - 前两种卷积 $ f(t - t_1) * g(t - t_2) = (f*g)(t - t_1 - t_2) $.
  - 第三种卷积 $ f\pqty{\dfrac{n}{n_1}} * g\pqty{\dfrac{n}{n_2}} = (f * g) \pqty{\dfrac{n}{n_1 n_2}} $. (记非整数处的取值为零)

第一种卷积还有如下性质:

- 微分 $ (f * g)' = f' * g $.
- 积分 $ (f*g)^{(-1)} = f^{(-1)} * (g) $.
- 等效 $ f * g = f^{(-1)} * g' $.

第二种卷积还有如下性质:

- 差分
  - 向后差分 $ \nabla (f * g) = (\nabla f) * g $.
  - 向前差分 $ \Delta (f * g) = (\Delta f) * g $.
- 求和 $ \dsum (f * g) = \pqty{\dsum f} * g $.
- 等效 $ f * g = \pqty{\dsum f} * \pqty{\nabla g} $.

### A.8  常用函数的傅里叶变换

- 记号约定

  - $ G_\tau(t) = \bb I \Bqty{
    	-\dfrac{\tau}{2} < t < \dfrac{\tau}{2}
    } = u\pqty{
    	t + \dfrac{\tau}{2}
    } - u\pqty{
    	t - \dfrac{\tau}{2}
    } $.

    $ G_\tau(t) = \dfrac{\sgn(\omega_0 + \omega) + \sgn(\omega_0 - \omega)}{2} $.

    注意是从 $ -\dfrac{\tau}{2} $ 到 $ \dfrac{\tau}{2} $.

    即 `HeavisidePi[t/\[Tau]]`

  - $ \sgn(t) = \bb I\Bqty{t > 0} - \bb I\Bqty{t < 0} = 2 u(t) - 1 $.

    即 `Sign[t]`

  - $ \delta_T(t) = \nsuminf \delta(t - n T) $.

  - $ T_\tau(t) = \pqty{1 - \vqty{\dfrac{t}{\tau}}} G_{2\tau}(t) $.

    注意是从 $ -\tau $ 到 $ \tau $.

    即 `HeavisideLambda[t\[Tau]]`

- 备注

  - 可以由时移、频移、尺度变换得出的性质, 不再列出 (比如复指数信号的傅里叶变换).
  - 默认 $ n, k \in \N $, 且表达式在有意义时成立.
  - 默认 $ T_0 = \dfrac{2\pi}{\omega_0}, \alpha > 0 $.

### A.9  常用单边拉普拉斯变换

如果引入定义合适的分数阶微积分，则常用的单边拉普拉斯变换归根结底，都可由 $ \forall n \in \R \colon \delta^{(n)}(t) \lt s^n $ 得到.

|      参数条件       |                       原函数 $ f(t) $                        |                     象函数 $ F(s) $                     | 收敛域                                                       |
| :-----------------: | :----------------------------------------------------------: | :-----------------------------------------------------: | :----------------------------------------------------------- |
|    $ n \in \Z $     |                     $ \delta^{(n)}(t) $                      |                         $ s^n $                         | 当 $ n \ge 0 $ 时为 $ s \in \C $；<br />当 $ n < 0 $ 时为 $ \Re(s) > 0 $. |
|      $ n > 0 $      |                         $ t^{n-1} $                          |               $ \dfrac{\Gamma(n)}{s^n} $                | $ \Re(s) > 0 $.                                              |
|    $ n \in \N $     |                           $ t^n $                            |                 $ \dfrac{n!}{s^{n+1}} $                 | $ \Re(s) > 0 $.                                              |
|        $ / $        |                           $ u(t) $                           |                    $ \dfrac{1}{s} $                     | $ \Re(s) > 0 $.                                              |
|  $ \alpha \in \R $  |                      $ \e^{-\alpha t} $                      |                 $ \dfrac{1}{s+\alpha} $                 | $ \Re(s) > -\alpha $.                                        |
| $ \omega_0 \in \C $ |             $ \e^{-\alpha t} \cos(\omega_0 t) $              |    $ \dfrac{s + \alpha}{(s+\alpha)^2 + \omega_0^2} $    | $ s \in \C $.                                                |
| $ \omega_0 \in \C $ |             $ \e^{-\alpha t} \sin(\omega_0 t) $              |     $ \dfrac{\omega_0}{(s+\alpha)^2 + \omega_0^2} $     | $ s \in \C $.                                                |
| $ \omega_0 \in \C $ |                    $ t\cos(\omega_0 t) $                     |   $ \dfrac{s^2 - \omega_0^2}{(s^2 + \omega_0^2)^2} $    | $ \Re(s) > 0 $.                                              |
| $ \omega_0 \in \C $ |                    $ t\sin(\omega_0 t) $                     |      $ \dfrac{2\omega_0 s}{(s^2 + \omega_0^2)^2} $      | $ \Re(s) > 0 $.                                              |
| $ \omega_0 \in \C $ |             $ \e^{-\alpha t} \cosh(\omega_0 t) $             |     $ \dfrac{s+\alpha}{(s+\alpha)^2 - \omega_0^2} $     | $ \vqty{\Re(s)} < \omega_0 $.                                |
| $ \omega_0 \in \C $ |             $ \e^{-\alpha t} \sinh(\omega_0 t) $             |     $ \dfrac{\omega_0}{(s+\alpha)^2 - \omega_0^2} $     | $ \vqty{\Re(s)} < \omega_0 $.                                |
|        $ / $        |                          $ \Sa(t) $                          |                      $ \arccot s $                      | $ \Re(s) \ge 0 $.                                            |
|        $ / $        |                          $ \Si(t) $                          |          $ \dfrac{1}{s} \arctan\dfrac{1}{s} $           | $ \Re(s) \ge 0 $.                                            |
| $ \omega_0 \in \C $ |               $ \dfrac{\sinh(\omega_0 t)}{t} $               | $ \dfrac{1}{2} \ln \dfrac{s + \omega_0}{s - \omega_0} $ | $ s > \vqty{\omega} $.                                       |
|    $ n \in \N $     | $  L_n(t) = \dfrac{\e^t}{n!} \ddv[n]{t} \pqty{t^n \e^{-t}}  $ |              $ \dfrac{(s-1)^n}{s^{n+1}} $               | $ \Re(s) > 1 $.                                              |



不存在拉普拉斯变换的函数：

- $ \dfrac{1}{t^n}, n \in \N^+ $.
- $ \dfrac{\cos t}{t} $.

---


$$
\begin{align}
\dfrac{u(t)}{(t+a)^n} &\lt \begin{cases}
	s^{n-1} \e^{as} \dint_{-\infty}^{-as} \dfrac{\e^x \dx}{(-x)^n},
	& a > 0, n \in \R,
	\\
	s^{n-1} \e^{as} \Gamma(1-n, as),
	& a \in \R, n < 1,
	\\
	不存在, & a \le 0, n \ge 1.
\end{cases}
\end{align}
$$

- 备注
  - 不能用幂级数展开法求解，因为不满足条件.
  - 右式的积分为柯西主值积分.
- 特殊的
  - 当 $ n \in \N $ 时，有 $ t^n u(t) \lt \dfrac{n!}{s^{n+1}} $.
  - 当 $ a > 0 $ 时，有 $ \dfrac{u(t)}{t+a} \lt -\e^{as} \operatorname{Ei}(-as) $.
