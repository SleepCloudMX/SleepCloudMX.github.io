设四边形四条边分别为复数 $ 2a, 2b, 2c, 2d $, 四个正方形的中心分别为 $ p, q, r, s $, 则有
$$
\renewcommand{\i}{\mathrm{i}}
\begin{cases}
p = (1 + \i) a \\
q = 2a + (1 + \i) b \\
r = 2a + 2b + (1 + \i) c \\
s = 2a + 2b + 2c + (1 + \i) d \\
\end{cases}
$$
设 $ A = s - q,\, B = r - p $, 欲证垂直且相等, 即证 $ B = \i A $, 或 $ A + \i B = 0 $, 代入得:
$$
A + \i B = (a + b + c + d) + \i (a + b + c + d) = 0
$$
证毕.
