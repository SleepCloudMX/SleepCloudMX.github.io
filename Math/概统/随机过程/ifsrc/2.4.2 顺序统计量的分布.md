0 < y_1 < y_1 + h

则有
$$
\begin{align}
% &\hspace{1.3em}
f(\soneto{y}{n})
&= \lim_{h \to 0} \dfrac{1}{h^n}
P\Bqty{
	\knsum \Bqty{

	}

\\
&= \lim_{h \to 0} \dfrac{1}{h^n}
\sum_{\soneto{i}{n}} P\Bqty{
	\knsum \Bqty{
		y_k < Y_{i_k} \le y_k + h
	}
} \\
&= \lim_{h \to 0} \dfrac{n!}{h^n} P\Bqty{
	\knsum \Bqty{
		y_k < Y_{i_k} \le y_k + h
	}
} \\
&= n! \inprod f(y_i).
\end{align}
$$
