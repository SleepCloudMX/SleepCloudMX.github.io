由 $ \xi_{n + 1} $ 与 $ X_0, X_1, \cdots, X_n $ 相互独立,
$$
\begin{align}
&\hspace{1.3em}
P\Bqty{
	X_{n + 1} = i_{n + 1} \mid
	X_0 = i_0, \cdots, X_n = i_n
} \\
&= P\Bqty{
	f(X_n, \xi_{n + 1}) = i_{n + 1} \mid
	X_0 = i_0, \cdots, X_n = i_n
} \\
&= P\Bqty{
	f(i_n, \xi_{n + 1}) = i_{n + 1}
} \\
&= P\Bqty{
	X_{n + 1} = i_{n + 1} \mid X_n = i_n
},
\end{align}
$$
于是 $ \Bqty{X_n,n \ge 0} $ 是马尔科夫链, 且 $ p_{ij} = P\Bqty{f(i, \xi_1) = j} $.
